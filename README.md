# ALBERTO Restaurant Website

A modern luxury restaurant website built with React, Vite, and Tailwind CSS, featuring server-side rendering with TanStack Start.

## 🍽️ Overview

ALBERTO is a premium restaurant web application showcasing bold flame-grilled burgers, velvet drinks, and golden desserts. The site combines luxury aesthetics with modern web technologies to deliver an elegant dining experience online.

## ✨ Features

- **Premium Design**: Deep maroon and gold color palette with cinematic gradients
- **Responsive Layout**: Fully optimized for mobile and desktop devices
- **Interactive Menu**: Category filtering and detailed item cards
- **Smooth Animations**: Custom transitions and micro-interactions
- **Theme System**: Dark/light mode toggle with persistence
- **Component Architecture**: Reusable React components with TypeScript

## 🛠️ Tech Stack

- **Frontend**: React 19 with TypeScript
- **Framework**: TanStack Start (SSR)
- **Routing**: TanStack Router
- **Styling**: Tailwind CSS v4
- **Build Tool**: Vite
- **State Management**: TanStack Query
- **Icons**: Lucide React

## 🚀 Getting Started

### Prerequisites

- Node.js 16+
- npm or yarn

### Installation

```bash
git clone <repository-url>
cd alberto-restaurant
npm install
```

### Development

```bash
npm run dev
```

### Production Build

```bash
npm run build
npm run preview
```

## 📁 Project Structure

```
src/
├── components/          # React components
│   ├── Navbar.tsx      # Navigation with scroll effects
│   ├── Hero.tsx        # Landing section
│   ├── FullMenu.tsx    # Interactive menu
│   └── Footer.tsx      # Restaurant info
├── routes/             # TanStack Router pages
```
